
# Secure Chat

You can chat with your friends and family.


## Features

- Realtime Chats
- Share Images
- Login with email and password
- Incredible UI
- Can delete sent messages instantly


## Tech Stack

**Architecture Pattern:** MVVM

**Server:** Firebase - Authentication, Realtime Database, Storage
